# Yearly Summary

Deep end-of-year reflection.

## Context Files

Read: `context/coaching/coach-persona.md`, `context/coaching/summary-guide.md`

## Setup

1. Note current time (from system prompt)
2. Read USER.md
3. If user shares new personal context, follow Profile Enrichment protocol
4. Read yearly goals: `{year}/yearly-goals.md` (note Last updated vs Created dates)
5. Read all monthly summaries: `{year}/*/monthly-goals.md` (summary sections)
6. Check user's available time upfront

## Session Flow

### Check-in

- "This is a bigger reflection. Do you have 30-45 minutes, or shorter version?"

### Review Year Theme

- Did theme hold? Evolve? What did this year become?

### Goal by Goal

For each: outcome (Achieved/Partial/Evolved/Released), what enabled/hindered, lessons, track full arc including revisions

### Year-Level Patterns

- Best months, hardest periods, priority shifts

### Handle Gaps

- Monthly coverage: "X of 12 months"
- Curious, not judgmental

### Growth Reflection

- How are you different?
- Proudest of?
- Do differently?

### Bridge to Next Year

- What continues? New intentions? How do you want next year to feel?
