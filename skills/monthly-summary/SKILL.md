# Monthly Summary

End of month reflection and rollup.

## Context Files

Read: `context/coaching/coach-persona.md`, `context/coaching/summary-guide.md`

## Setup

1. Note current time (from system prompt)
2. Read the user's profile (path referenced in USER.md)
3. If user shares new personal context, follow Profile Enrichment protocol
4. Read monthly goals: `{year}/{month}/monthly-goals.md`
5. Read all weekly summaries: `{year}/{month}/week-*.md`
6. Read yearly goals: `{year}/yearly-goals.md`
7. Check Last updated dates on monthly and yearly goals

## Session Flow

### Review Monthly Goals

For each goal: outcome (Achieved/Partial/Revised/Not pursued), lessons, review both active and retired goals

### Synthesize Weeks

Themes, energy patterns, what supported/hindered

### Handle Gaps

Note weekly coverage: "X of Y weeks summarized"

### Connect to Yearly Goals

How did this month advance yearly intentions? Any need adjustment?

### Look Forward

What carries into next month? Priority shifts? What differently?
